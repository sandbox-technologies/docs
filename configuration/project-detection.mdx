## Package Managers

Detected from lock files:

| Lock File | Manager |
|-----------|---------|
| `package-lock.json` | npm |
| `yarn.lock` | Yarn |
| `pnpm-lock.yaml` | pnpm |
| `bun.lockb` | Bun |

## When Detection Fails

Create `.inspector.json`:

```json
{
  "setup": "npm install",
  "run": "npm run dev"
}
```

## Node.js Version Priority

1. `.inspector.json` runtimes.node
2. `.nvmrc`
3. `.node-version`
4. `package.json` engines
