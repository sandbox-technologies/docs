## Package Managers

Detected from lock files:

| Lock File | Manager |
|-----------|---------|
| `package-lock.json` | npm |
| `yarn.lock` | Yarn |
| `pnpm-lock.yaml` | pnpm |
| `bun.lockb` | Bun |

## When Detection Fails

Create an `inspector.json` at your project root. Only `dev` is required:

```json
{
  "dev": "npm run dev"
}
```

Add `install` if your project needs a setup step (e.g. `npm install`):

```json
{
  "install": "npm install",
  "dev": "npm run dev"
}
```

See [inspector.json reference](/configuration/inspector-json) for the full schema.

## Node.js Version Priority

1. `inspector.json` `node` field
2. `.nvmrc`
3. `.node-version`
4. `package.json` engines
