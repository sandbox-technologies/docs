## Clone a Repository

1. Click **GitHub** on start screen
2. Enter `owner/repo`
3. Choose location and clone

## Create a Pull Request

1. Commit and push your changes
2. Click **Create Pull Request** in Publish panel
3. Fill in title and description
4. Submit

## Checkout a PR

1. Open branch selector
2. Click **Pull Requests** tab
3. Select the PR to review

## Deep Links

Open repos directly with URL schemes:

```bash
inspector://open?repo=owner/repo&branch=main
inspector://open?repo=owner/repo&pr=123
```
