## Actions

| Action | How |
|--------|-----|
| **Move** | Drag element |
| **Edit text** | Double-click |
| **Nudge** | Arrow keys (Shift = 10px) |
| **Delete** | Select + Delete key |

## Workflow

1. Enable Move Tool (`V`)
2. Make your edit (drag, double-click, etc.)
3. Click **Apply** to send to AI agent
4. AI updates your code
