Select an element to open the Design Panel. Press `D` to toggle.

## Sections

| Section | Properties |
|---------|------------|
| **Position** | X, Y, rotation, scale |
| **Auto Layout** | Flex direction, gap, padding |
| **Typography** | Font, size, weight, line height |
| **Fill** | Background color, gradients |
| **Stroke** | Border width, style, color |
| **Effects** | Shadows, filters |

## Usage

1. Select an element
2. Adjust values in Design Panel
3. See live preview
4. Click **Apply** to save

<Tip>
  Changes are implemented using your project's existing patterns (CSS, Tailwind, etc.).
</Tip>
