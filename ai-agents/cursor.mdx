## Setup

1. Install [Cursor](https://cursor.sh)
2. Open Inspector **Settings**
3. Click **Connect Cursor**
4. Approve in Cursor when prompted

## How It Works

Inspector sends visual edits to Cursor's AI, which updates your code files. Changes hot-reload automatically.

## Tips

- Create a `.cursorrules` file for project-specific AI guidance
- Keep Cursor running while using Inspector
- Model selection is controlled in Cursor's settings
