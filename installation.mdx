## Install

1. Download the `.dmg` from [tryinspector.com](https://tryinspector.com)
2. Drag Inspector to Applications
3. Launch and connect an AI agent in Settings


## System Requirements

| Requirement | Details |
|-------------|---------|
| **OS** | macOS 12+ |
| **RAM** | 8 GB minimum |
| **AI Agent** | Cursor, Claude Code, or Codex subscription |
